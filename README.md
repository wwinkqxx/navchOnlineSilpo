# 🛒 Сильпо Каса (Silpo Cash Register)

Консольна програма для моделювання роботи каси в супермаркеті «Сільпо».  
Дозволяє створювати замовлення, керувати кошиком, обирати час доставки, оплачувати та отримувати квитанції у вигляді текстових файлів.

## 🚀 Можливості

- 📋 Перегляд списку доступних товарів
- ➕ Додавання товарів до кошика
- ✏️ Зміна кількості товару
- ❌ Видалення товару з кошика
- 📅 Встановлення дати, часу та адреси доставки
- 💳 Оплата замовлення (з імітацією успіху/невдачі)
- ❌ Скасування замовлення
- 💾 Автоматичне збереження квитанцій у текстові файли
- 🕒 Імітація процесу доставки після оплати

## 🧰 Технології

- C++17
- Стандартна бібліотека (STL)
- Робота з файлами
- Багатопотоковість (`<thread>`, `<chrono>`)

## 📁 Структура проєкту

- `main.cpp` – основний файл з реалізацією
- `order_*.txt` – автоматично згенеровані файли з деталями замовлення
